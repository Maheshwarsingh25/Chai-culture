# Chai Culture – Coming Soon

A simple, responsive “Coming Soon” landing page created for Chai Culture — a premium instant chai brand inspired by royal Indian traditions.

## About
Chai Culture celebrates the warmth, heritage, and elegance of traditional Indian chai. This page is designed to introduce the brand while building anticipation for the launch.

## Built With
HTML, CSS, and JavaScript

## Features
- Brand logo and tagline
- Short brand introduction
- Launching soon indicator
- Email notification signup
- Social media links
- Mobile-responsive layout

## Design Notes
The design uses warm, royal tones and elegant typography to reflect heritage and authenticity while keeping the layout minimal and modern.
